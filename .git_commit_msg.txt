重构：全面重组项目结构并实现模块化

主要变更：

1. 构建系统
   - 移除 Makefile，改用 CMake (CMakeLists.txt)
   - 新增 run_all_tests.sh 测试脚本

2. 目录结构重组
   - src/backend/: 后端模块（编译器、虚拟机、字节码）
     * compiler/: 编译器及优化模块（fusion、inline、peephole）
     * vm/: 虚拟机核心及上下文管理
     * bytecode/: 字节码生成和调试
   
   - src/frontend/: 前端模块（词法、语法、AST）
     * lexer/: 词法分析
     * parser/: 语法分析及 OOP/类型解析
     * ast/: 抽象语法树
   
   - src/runtime/: 运行时系统
     * closure/: 闭包、函数原型、upvalue
     * objects/: 运行时对象（数组、类、实例、映射、方法、字符串）
   
   - src/utils/: 工具模块
     * 哈希表、作用域管理等工具
   
   - tests/: 测试系统
     * framework/: 测试框架（xtest.h）
     * unit/: 单元测试（包含上下文和错误处理测试）

3. 新增功能
   - 编译器上下文管理 (compiler_context)
   - 虚拟机上下文管理 (vm_context)
   - 统一错误处理 (xerror)
   - 编译器优化：指令融合(fusion)、内联(inline)、窥孔优化(peephole)

4. 代码清理
   - 删除过时文件：xcompiler.c, xstate.c, xvalue.c, xmem.c 等
   - 删除冗余头文件：xeval.h, xgc.h, xobject.h
   - 统一头文件到 include/xray/

5. 文档和配置
   - 更新 README.md 和 BUILD.md
   - 配置 .gitignore 仅保留 BUILD.md 和 README.md

此次重构使代码结构更加清晰、模块化，为后续开发和维护奠定了良好基础。
